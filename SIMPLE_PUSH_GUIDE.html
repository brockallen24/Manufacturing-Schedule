üîê GitHub Authentication Help

If It Asks for Password:

GitHub requires a Personal Access Token instead of your password.

Quick Setup:

  1. Go to: Create Token ‚Üí
  2. Click Generate new token (classic)
  3. Give it a name: "Manufacturing Schedule"
  4. Check the box: repo (full control)
  5. Click Generate token at bottom
  6. Copy the token (it looks like: ghp_xxxxxxxxxxxx)
  7. Use THIS as your "password" when pushing
